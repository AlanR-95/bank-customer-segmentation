# Aplicación de machine learning para la segmentación de clientes de un banco.

Aplicando herramientas de machine learning (K-means y Affinity Propagation) para realizar la segmentación de los clientes del banco.

- Limpieza de datos
- Análisis exploratorio de datos
- Aplicación del algoritmo Kmeans para la segmentación, hallando 3 grupos (clusters) representativos de los clientes del banco.
- Aplicación del algoritmo AF, hallando 4 grupos (clusters) representativos.
